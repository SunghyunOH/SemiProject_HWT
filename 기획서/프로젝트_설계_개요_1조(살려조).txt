조 이름 : 살려조

프로젝트명 : HelloWorld Talk

구성원 별 역할 :

1. 팀장 고윤석 -  검색, 채팅
2. 팀원 김  호 - 회원가입, 로그인 창
3. 팀원 오성현 - 관리자페이지
4. 팀원 장정호 - 회원 수정, 마이페이지
5. 팀원 최창환 - HEADER, FOOTER, 메인페이지


개요 : 

 국내 다양한 언어를 사용하는 개발자와 의견을 주고 받고 학습도 할 수 있는 
 소통의 장이 되는 개발 커뮤니티 사이트를 기획

구현 기능 :

회원 관리 기능 : 로그인, 로그아웃, 회원가입
실시간 라이브 게시판 기능 : ALL게시판, 학습게시판, 질문게시판, 자유게시판, 공지게시판, 마이게시판, 댓글 기능
검색 기능, 채팅 기능,
관리자 기능 : 회원 관리, 신고 관리, 고객 문의 관리


설계의 주안점 :

수업 시간에 배운 여러가지 프로그램(자바, 오라클, HTML, CSS, JSP, 깃허브, 자바스크립트 J쿼리) 등
을 사용하여 학습, 복습을 하여 다양한 기능의 웹 사이트 구현을 할 수 있다.


사용기술 및 개발 환경 :

OS : Windows 10
Developer-Tool : VSCode / Eclipse / IntelliJ /SQLDeveloper
Front-End : HTML / CSS / Java Script / jQuery
Back-End : Java / Oracle DBMS / tomcat
